# p0-w3-d1-t5 · truthy и falsy

## Задача

В условии `if (значение)` JS смотрит не на «истину/ложь», а на truthy/falsy. Falsy-значений ровно восемь, остальное truthy, включая неочевидное вроде `'0'` или пустого массива.

```
falsy (8 штук): false 0 -0 '' null undefined NaN 0n
всё остальное — truthy (даже '0', [], {})

if (значение) — проверяет именно truthy/falsy
```

## Что нужно сделать

В `truthy.js` прогнать через `if`/`Boolean()` набор значений и определить truthy/falsy: `0`, `''`, `'0'`, `null`, `undefined`, `NaN`, пустой массив, пустой объект, пробел `' '`, `-1`. Сначала прогноз, потом проверка.

## Критерии приёмки

* Для каждого значения определено truthy/falsy с прогнозом до проверки.
* Отдельно отмечены ловушки: `'0'`, пустой массив, пустой объект, пробел — все truthy.
* Можешь перечислить все восемь falsy-значений по памяти.
